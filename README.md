# FHE Pinturas 🎨

A landing page profissional e moderna da **FHE Pinturas**, focada em serviços de pintura e acabamentos de alto padrão em Formosa, Goiás. O projeto foi construído para entregar máxima performance, um design premium e um SEO perfeitamente otimizado para liderar as buscas locais.

**URL do Projeto:** [https://fhepinturas.vercel.app](https://fhepinturas.vercel.app)

## 🚀 Tecnologias e Ferramentas

- **React 18**: Biblioteca JavaScript para construção da interface.
- **Vite**: Bundler ultra-rápido para desenvolvimento moderno.
- **Tailwind CSS v3**: Framework CSS utilitário para a estilização ágil e responsiva.
- **GSAP (GreenSock)**: Biblioteca profissional de animações, usada na fluidez de componentes visuais avançados.
- **Embla Carousel**: Carrossel de alta performance e suporte touch/drag perfeito para exibir a galeria de projetos.
- **TanStack Router**: Roteamento robusto e tipado.
- **Lucide React**: Biblioteca de ícones moderna e leve.

## 🌟 Principais Recursos e Diferenciais

- **Design Premium ("Pro Max"):** Backgrounds imersivos com efeito "Glassmorphism", gradientes sutis, e bordas estilizadas em SVG para divisão de seções.
- **SEO Otimizado:** Pontuação máxima nos relatórios do Lighthouse. Meta tags, imagens Open Graph, Twitter Cards, e Favicons totalmente configurados para garantir o 1º lugar no Google na região (Formosa/GO).
- **Responsividade 100%:** A interface se adapta milimetricamente a qualquer tamanho de tela, com navegação fluida em dispositivos móveis e otimização visual para telas ultralargas.
- **Animações Fluidas:** O GSAP controla animações de rolagem (ScrollReveal), e uma animação em plano de fundo altamente customizada no Call to Action (Contato), sem prejudicar a taxa de quadros (FPS).
- **Ações Rápidas Flutuantes (FABs):** Integração instantânea para ligação direta (`tel:`) ou contato via WhatsApp (`wa.me`), posicionados de forma acessível.

## 🛠️ Como Executar Localmente

### 1. Pré-requisitos
Certifique-se de que o **Node.js** (versão 18 ou superior) está instalado em sua máquina.

### 2. Instalação
Clone este repositório e instale as dependências:
```bash
# Clone o repositório
git clone https://github.com/seu-usuario/fhe-pinturas.git

# Entre no diretório
cd fhe-pinturas

# Instale as dependências
npm install
```

### 3. Rodando o servidor de desenvolvimento
Execute o comando abaixo para subir a aplicação localmente:
```bash
npm run dev
```
Acesse `http://localhost:5173` no seu navegador para ver o site funcionando.

### 4. Build para Produção
Para compilar os arquivos minimizados e otimizados:
```bash
npm run build
```
O código compilado estará na pasta `dist/`.
